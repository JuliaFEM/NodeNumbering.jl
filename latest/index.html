# NodeNumbering.element_adjacencies — Constant.

element_adjacencies

The Dict shows some element types and their node adjacencies. For example the element type Quad4 has four nodes/element so that node No. 1 is adjacent to nodes 2 and 4 etc.

# NodeNumbering.node_degrees — Method.

node_degrees(adjacency::Dict{Int, Vector{Int}})

Calculate the degrees for the adjacency graph nodes. Node degree = number of nodes a certain node is adjacent to.

Examples

julia> adjacency = Dict{Int, Vector{Int}}(
                   1 => [3, 8, 9],
                   2 => [3, 8, 7],
                   3 => [1, 2],
                   4 => [8, 9],
                   5 => [7, 8],
                   6 => [2, 7],
                   7 => [5, 2, 6],
                   8 => [1, 2, 4, 5],
                   9 => [1, 4]);

julia> node_degrees(adjacency::Dict{Int, Vector{Int}})
Dict{Int64,Int64} with 9 entries:
  7 => 3
  4 => 2
  9 => 2
  2 => 3
  3 => 2
  5 => 2
  8 => 4
  6 => 2
  1 => 3

# NodeNumbering.RCM — Method.

RCM(adjacency::Dict{Int, Vector{Int}}, degrees::Dict{Int, Int}, P::Int)

# NodeNumbering.create_adjacency_graph — Method.

create_adjacency_graph(elements::Dict{Int, Vector{Int}}, element_types::Dict{Int, Symbol})

Create a Dict that shows all nodes and their adjacencies.

Examples

julia> elements = Dict(
                  1 => [9, 1, 8, 4],
                  2 => [1, 3, 2, 8],
                  3 => [8, 2, 7, 5],
                  4 => [2, 6, 7]);

julia> element_types = Dict(
                       1 => :Quad4,
                       2 => :Quad4,
                       3 => :Quad4,
                       4 => :Tri3);

julia> create_adjacency_graph(elements, element_types)
Dict{Int64,Array{Int64,1}} with 9 entries:
  7 => [2, 6, 5]
  9 => [1, 4]
  4 => [9, 8]
  2 => [6, 7, 3, 8]
  3 => [1, 2]
  8 => [1, 2, 5, 4]
  5 => [8, 7]
  6 => [2, 7]
  1 => [3, 8, 9]
